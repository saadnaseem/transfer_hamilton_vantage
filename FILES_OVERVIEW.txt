CONFIGURATION (IF NEEDED)
================================================================================

To change parameters, edit user_params in transfer.py:

  user_params = {
      'well_volume': 1500,          # Total µL per destination well
      'source_well_volume': 9000,   # Max usable µL per source well
      'dead_volume': 100,           # Unaspiratable µL per source well
      'min_transfer_volume': 5.0,   # Minimum accurate transfer (µL)
      'max_tip_volume': 200.0,      # Threshold for splitting (µL)
      'culture_factor': 100,        # Culture dilution factor
      'wells_per_plate': 48,        # 48 or 96
      'plate_format': '48-well',    # '48-well' or '96-well'
      'add_culture_to_all_wells': True,
      'water_source': {
          'plate': 's_water',
          'well': 'A1',
          'type': 'plate'
      }
  }

Common adjustments:
  • Change well_volume to different total volume
  • Change wells_per_plate to 96 for larger plates
  • Change plate_format to match your equipment
  • Adjust min_transfer_volume if you have higher precision pipetting

  → Expected! MOPS concentration in target data is too small (0.0428 mM)
  → Would require 0.032 µL, below 5 µL minimum
  → MOPS is typically handled as a fixed component separately

Problem: Source depletion warnings
  → MgCl2 and NaCl need more stock
  → Either: Add more stock, or reduce target concentrations

Problem: Tests failing
  → Verify input CSV files are in data/ folder
